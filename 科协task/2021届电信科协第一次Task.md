1. 学会使用格式化输入输出函数 `printf()` 和 `scanf()` 等，自行查资料认识 `sprintf()` 和 `sscanf()` 。

题目：写出下列程序的运行结果，并思考为什么会是这种结果：

```c
#include <stdio.h>

int main(){
    int x=017,a=12,b=34;
    printf("%d\n", x);
    printf("%o\n", x);
    printf("%#o\n", x);
    printf("%x\n", x);
    printf("%#x\n", x);
    printf("%d", a, b);
    return 0;
}
```

1. 有见过 `%p` 这个占位符么？请自行查阅资料了解，并用它来验证一些数据类型相关的问题。
